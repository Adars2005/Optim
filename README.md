## Setup
1. Clone the repo
2. Run `flutter pub get`
3. Copy `.env.example` → `.env` and fill in your Firebase credentials
4. Add your own `google-services.json` in `android/app/`

6. Run the app with `flutter run`



OPTIM Description 

Optim is an AI-driven, end-to-end eye-care platform building the future of vision care in a single application—from eye scan to diagnosis, prescription, clinical reporting, and hospital integration in seconds. Powered by multimodal AI for retinal image analysis and visual question answering, Optim delivers instant, accurate diagnostic insights, replacing hours of waiting with downloadable digital reports. With dedicated portals for patients, ophthalmologists, and hospitals, the platform enables continuous patient monitoring, streamlines treatment workflows, and advances early intervention. Our mission is simple: faster treatment, smarter care, and preventing vision loss—especially for children and teens.
